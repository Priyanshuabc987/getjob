# **App Name**: PrepLinc

## Core Features:

- User Authentication & Profiles: Secure user registration and login. Create personalized student profiles showcasing skills, interests, and an automatically generated public portfolio based on completed tasks and projects.
- Dynamic Activity Feed: A real-time, personalized feed displaying micro-internship tasks, project updates from collaborators ('building in public'), and problems posted by startups, with prominent 'Start Task' and 'Join Project' call-to-action buttons.
- Micro-Internship & Task Marketplace: Browse a list of micro-internship tasks with descriptions, payment details, time commitment, and required skills. Users can filter by skill, time, and payment range, and apply/start tasks with a single click.
- Project Collaboration Hub: Students can create project listings (title, idea, skills needed), post updates on their progress, and break down projects into smaller, assignable micro-tasks. Other users can instantly join and contribute to specific tasks.
- AI-Powered Smart Connections & Discovery: An AI tool that suggests relevant micro-internships, collaboration projects, or peer connections based on a user's profile (skills, interests) and past platform activity to foster engagement and networking.
- Startup Portal & Challenge Posting: A streamlined interface for verified startups to post micro-internship tasks and real-world problems/challenges, including rewards and deadlines, providing essential opportunities for students.
- Gamified Engagement & Progress Tracking: Implement gamification elements such as badges for task completion, skill development, and collaboration. Track individual progress, earnings, and project contributions which automatically populate the user's public portfolio.

## Style Guidelines:

- The visual design takes inspiration from the vibrant energy of collaboration and the clarity of a learning platform. We use a rich, digital purple (#6626D9) as the primary brand color, representing creativity and innovation. This color contrasts effectively with lighter UI elements.
- A very light, desaturated purple (#F4F3F8) serves as the primary background color, providing a clean, spacious, and unintrusive canvas reminiscent of a digital notebook. Its subtle hue ties back to the primary color, maintaining visual harmony.
- A bright and energetic blue (#409FF0) is used as an accent color for key interactive elements like calls-to-action and important highlights. Its analogous relationship to the primary purple creates a cohesive yet dynamic palette that promotes action and engagement.
- Headlines will utilize 'Space Grotesk' (sans-serif), a contemporary font that provides a tech-forward and clean aesthetic, suitable for grabbing attention in a dynamic feed.
- Body text and other readable content will use 'Inter' (sans-serif), a highly legible and neutral font that ensures optimal readability across various screen sizes and content types, from task descriptions to project updates.
- Modern, clean, and highly recognizable vector icons should be used throughout. Focus on clarity for actions (e.g., 'Start Task', 'Join Project') and utilize elements that reflect community (like Discord-style user groups) and interaction (like Instagram's likes/shares/comments).
- The design will adopt a mobile-first approach, prioritizing responsive layouts and touch-friendly components. A card-based design for feed items, micro-internships, and projects will create an Instagram-like scrollable experience. Navigation will be intuitive, likely utilizing a sticky bottom bar on mobile and a clear sidebar on larger screens, reflecting a Notion-style organization.
- Subtle, fast, and satisfying micro-animations will be employed to enhance user feedback and engagement. This includes smooth transitions between screens or content loads, lively effects for successful interactions (e.g., submitting a task), and engaging animations for gamified elements like badge unlocks.
